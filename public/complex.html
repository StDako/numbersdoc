numbers.complex

numbers.complex is actually a functional object and not a set of functions. An example of a Complex type is Z = Complex(1,1). This section describes the methods associated with the Complex type.

Complex.add(z)

Add z to this complex number.

Parameters

z : Complex

A complex number.
Returns

w : Complex

A new complex number.
Errors

This function does not raise any errors.
